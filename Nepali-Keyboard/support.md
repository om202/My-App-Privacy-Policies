# Support — Nepali Keyboard

Thank you for using **Nepali Keyboard** for macOS! If you need help, have a question, or want to report a bug, we're here for you.

## Getting Started

1. Open **System Settings** → **Keyboard** → **Input Sources**
2. Click **Edit...**, then tap the **+** button
3. Search for **Nepali Keyboard** and add it
4. Use **Control + Space** to switch between input sources

For a full typing guide, open the app from your menu bar and visit the **Guide** tab.

## Common Issues

### Keyboard not appearing in Input Sources
Make sure the app is running. Look for the **ने ⌨️** icon in your menu bar. If it's not there, open Nepali Keyboard from your Applications folder.

### Typing produces English instead of Nepali
Ensure Nepali Keyboard is selected as your active input source. Click the input menu in the menu bar and select **Nepali Keyboard**, or press **Control + Space** to toggle.

### App not starting after installation
Try logging out and back in, or restart your Mac. macOS sometimes needs a refresh to register new input methods.
