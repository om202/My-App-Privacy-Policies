# RapidPhoto Support

**Fast batch photo editing for Mac**

---

## Getting Started

### How do I import images?
1. Click **Import Images** in the toolbar
2. Select up to 100 images from your Mac
3. Images will appear in the sidebar

### How do I crop images?
1. Select an image from the sidebar
2. Choose an aspect ratio (16:9, 1:1, etc.)
3. Adjust the crop area by dragging
4. Click **Apply to All** to use the same crop for all images

### How do I export images?
1. Make sure all images have crop settings applied
2. Click **Execute Batch** in the toolbar
3. Choose a save location
4. Images will be exported to a timestamped folder

## Troubleshooting

### Images won't import
- **Check file format:** Supported formats are JPEG, PNG, HEIC, GIF, BMP, TIFF
- **Check file size:** Maximum 100MB per image
- **Check image dimensions:** Maximum 16,384 x 16,384 pixels

### Export failed
- **Check disk space:** Make sure you have enough free space
- **Check permissions:** Grant file access when prompted
- **Try a different location:** Export to your Desktop or Documents folder

### Subscription not working
- **Restore purchases:** Click "Restore Purchases" in the subscription dialog
- **Check internet:** Make sure you're connected to the internet
- **Restart the app:** Quit and relaunch RapidPhoto

### App crashes or freezes
- **Update macOS:** Make sure you're running macOS 14.0 or later
- **Restart your Mac:** A simple restart often fixes issues
- **Reduce batch size:** Try processing fewer images at once

## Privacy

RapidPhoto processes all images **locally on your Mac**. We do not:
- Upload your images to any server
- Collect or store your photos
- Share your personal data with third parties

We collect **anonymous usage analytics** to improve the app:

- Number of images processed (not the images themselves)

- **Not linked to your identity**
